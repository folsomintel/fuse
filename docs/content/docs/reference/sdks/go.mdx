## Install

```bash
go get github.com/folsomintel/fuse/sdks/go
```

## Quickstart

```go
package main

import (
	"context"
	"log"

	fuse "github.com/folsomintel/fuse/sdks/go"
)

func main() {
	client, err := fuse.New("https://orchestrator.example.com", "token")
	if err != nil {
		log.Fatal(err)
	}

	env, err := client.Environments.Create(context.Background(), fuse.CreateRequest{
		TaskID: "t-1",
		Spec:   fuse.Spec{CPUs: 2, RamMB: 2048},
	})
	if err != nil {
		log.Fatal(err)
	}
	log.Println(env.ID, env.State)
}
```

## Client construction

```go
func New(baseURL, token string, opts ...Option) (*Client, error)
```

`baseURL` must be a non-empty, parseable URL. `token` is sent as a bearer token and
may be empty for orchestrators running without auth. Functional options:

| Option | Purpose |
| --- | --- |
| `WithHTTPClient(*http.Client)` | Override the HTTP client used for normal (non-streaming) requests. |
| `WithUserAgent(string)` | Override the `User-Agent` header (default `fuse-go/<version>`). |
| `WithRequestID(func() string)` | Generator called once per request for `X-Request-ID`; empty return omits the header. |
| `WithBaseURL(string)` | Override the base URL passed to `New`. |

`Client` groups four services that share one transport:

```go
type Client struct {
	Environments *EnvironmentsService
	Snapshots    *SnapshotsService
	Hosts        *HostsService
	APIKeys      *APIKeysService
}
```

## Environments

```go
func (s *EnvironmentsService) List(ctx context.Context, opt ListEnvironmentsOptions) ([]EnvironmentInfo, error)
func (s *EnvironmentsService) Get(ctx context.Context, vmID string) (*EnvironmentInfo, error)
func (s *EnvironmentsService) Create(ctx context.Context, reqBody CreateRequest) (*EnvironmentInfo, error)
func (s *EnvironmentsService) Drain(ctx context.Context, vmID string) (*EnvironmentInfo, error)
func (s *EnvironmentsService) Fork(ctx context.Context, vmID string, opts ForkOptions) (*EnvironmentInfo, error)
func (s *EnvironmentsService) RotateToken(ctx context.Context, vmID string) error
func (s *EnvironmentsService) Destroy(ctx context.Context, vmID string) error
func (s *EnvironmentsService) Events(ctx context.Context, vmID string) (<-chan Event, error)
```

`Events` opens the SSE stream and returns a channel of `Event` values, closed when
the stream ends (cleanly on EOF, or after a final `Event` with `Err` set on
failure). It uses a no-timeout client: cancel `ctx` to stop it.

## API keys

```go
func (s *APIKeysService) Create(ctx context.Context, label string) (*CreatedAPIKey, error)
func (s *APIKeysService) List(ctx context.Context) ([]APIKey, error)
func (s *APIKeysService) Revoke(ctx context.Context, id string) error
```

`Create` requires the master token; the server enforces this. `CreatedAPIKey.Key`
holds the raw secret, returned once and unrecoverable afterward.

## Errors

All non-2xx responses surface as an error from `CheckResponse` carrying the
orchestrator's structured error code (`not_found`, `conflict`, `invalid_argument`,
`unavailable`, `internal`); see [HTTP API overview](/docs/reference/api) for the
full error envelope shape.
