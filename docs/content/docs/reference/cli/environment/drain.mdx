---
title: fuse environment drain
description: Gracefully drain (phase one of teardown)
icon: LogOut
---

```bash
fuse environment drain <id>
```

No flags beyond the globals. Signals the guest agent to stop gracefully;
transitions the environment from `running` to `draining`. Only running
environments can be drained, any other state returns a conflict error.

Drain is phase one of a two-phase teardown, it does **not** destroy the
environment. The CLI prints a reminder of this after a successful drain. Follow
up with [`fuse environment destroy`](/docs/reference/cli/environment/destroy)
to actually remove it:

```bash
fuse environment drain fuse-abc123
# draining environment "fuse-abc123"
# drain is phase-1 only; it does not auto-destroy. run
# `fuse environment destroy fuse-abc123` to remove it

fuse environment destroy fuse-abc123
```
